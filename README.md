## ✨ Key Features

* **🌓 Adaptive Theming**: Integrated Dark/Light mode toggle with `localStorage` persistence.
* **🤖 3D Hero Section**: Features an interactive 3D robot background powered by **Spline**.
* **📱 Fully Responsive**: Optimized for everything from mobile screens to 4K monitors.
* **🎭 Smooth Animations**: Utilizes the `IntersectionObserver` API for sleek, high-performance fade-in effects on scroll.
* **📊 Dynamic Portfolio**: Organized sections for Education, Experience, and high-impact Technical Projects.
